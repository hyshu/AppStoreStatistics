App statistics on the App Store
----

App Store 内のiOSアプリで使用されているフレームワークを、公開年ごとに調査した統計です。

現在は日本の App Store だけを対象にしています。今のところ他の国の統計の作成は考えていません。

今回の調査によって統計以外で得られた有用な情報は、Zennで公開しています。

https://zenn.dev/hyshu
